# Crypto currency tg bot
![Node.js Version][node-version-image]

Send _/prices_ to chat with your bot and get the current cryptocurrency rates.

Copy and configure env's:
```bash
cp .env.example .env
```
Install dependencies:
```bash
npm i
```
Start application:
```bash
npm start
```
